Organisation of the Military:

  • Ημιομάδα -> Firearm (~5)
  • Ομάδα -> Squad (~10)
  • Διμοιρία -> Platoon (~30)
  • Λόχος -> Company (100)
  • Τάγμα -> Battalion (300)
  • Σύνταγμα -> Regiment (1000)
  • Ταξιαρχία -> Brigade (3.000)
  • Μεραρχία -> Division (10.000)
  • Σώμα Στρατού -> Corps (30.000)
  • Στρατιά -> Army (100.000)
  • Ομάδα Στρατιών -> Army Group (400.000)
  • Θέατρο -> Theater (1.000.000)
